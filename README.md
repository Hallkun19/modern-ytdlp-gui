# Modern Downloader (ModernDL)

Modern Downloader は、`yt-dlp` をシンプルで使いやすくしたオープンソースのビデオダウンローダーです。
多分だいぶ軽量で高速に動作します！

## 🚀 主な機能

- **クイックモード**: URL を貼り付けるだけで即座にダウンロード可能。
- **キューモード**: 複数の動画を一括で管理・ダウンロード。
- **高度なオプション**:
  - 最高画質 (8K/4K) から低画質までの選択。
  - ビデオコーデック (H.264, VP9, AV1) およびオーディオコーデックの指定。
  - ダウンロード範囲 (開始時間・終了時間) の指定。
  - メタデータおよびサムネイルの埋め込み。
- **マルチ言語対応**: 日本語、英語、韓国語、中国語をサポート。
- **テーマ切り替え**: ライトモードとダークモードに対応。

## 🛠 インストール / ビルド手順

### 事前準備

1. **Rust と Node.js**: システムにインストールされている必要があります。
2. **サイドカーの準備**: `yt-dlp` および `ffmpeg` のバイナリを `src-tauri/bin/` ディレクトリに配置する必要があります（セキュリティとライセンスの観点からリポジトリには含まれていません）。

### 開発モードの起動

```bash
npm install
npm run tauri dev
```

### 製品ビルド

```bash
npm run tauri build
```
